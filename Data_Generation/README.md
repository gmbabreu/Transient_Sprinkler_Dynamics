# Laser Sprinkler Angular Displacement Analysis

MATLAB scripts for extracting angular displacement data from video recordings of a laser sprinkler experiment. The pipeline processes `.mov` trial videos, isolates the laser position frame-by-frame using color masking, converts pixel positions to angular displacement, and plots aligned trial data.

---

## Files

### `generateData.m`
Processes raw video files for each trial and outputs a tab-delimited `.txt` data file per trial.

**Inputs:** `.mov` video files (`1500f1.mov`, `1500f2.mov`, `1500f3.mov`) and pre-extracted frame directories (`1500f1_frames/`, etc.)

> **Note on frame extraction:** Frame extraction is commented out by default (lines 33–47). Uncomment that block the first time you run on a new machine to generate the frame directories. It takes time and only needs to be done once.
> 
**Steps:**
0. Extract frames if not done so already.
1. Reads frames from the trial's frame directory.
2. Applies `ycbcrMask` to isolate the laser in each frame.
3. Computes a weighted mean x-position of the laser (weighted by color magnitude).
4. Normalizes position relative to the first frame (x = 0).
5. Converts pixel displacement → inches → angular displacement (degrees) using:
   - `fixed_distance = 49` inches (sprinkler mirror to ruler board)
   - `inches_per_pixel = 28/1920`
   - `theta = 0.5 * atan2(inches, fixed_distance)`
6. Writes `[time, theta]` to `1500fN_data.txt`.


Each trial also produces a `1500fN_maskedData.mp4` showing the masked frames.

---

### `plotData.m`
Reads the `.txt` data files produced by `generateData.m`, aligns the trials in time, computes a mean, and plots everything.

**Steps:**
1. Loads all trial data and pads mismatched-length arrays with `NaN` so they can be combined.
2. Estimates a time derivative to find when each trial's sprinkler starts moving (first derivative > 1).
3. Shifts all trials to a common start time based on the earliest onset.
4. Plots each shifted trial and their mean over the window `t = [0, 75]` seconds.
5. Saves mean data to `forward_1500_meanData.csv`.

> **Open question in the code:** Time alignment currently uses the derivative onset heuristic. A comment suggests using audio from the video to detect pump start as a potentially more accurate alternative.

---

### `ycbcrMask.m`
Auto-generated by MATLAB's **Color Thresholder app** (generated 04-Jun-2026). Isolates the laser dot from a frame by thresholding in YCbCr color space.

| Channel | Min | Max |
|---------|-----|-----|
| Y (luma) | 146 | 221 |
| Cb | 110 | 124 |
| Cr | 138 | 165 |

Returns a binary mask `BW` and a masked RGB image with the background zeroed out.

---

## Running the Pipeline

1. Place `.mov` files in the working directory.
2. On first run, uncomment lines 33–47 in `generateData.m` to extract frames into subdirectories.
3. Run `generateData.m` — produces `1500fN_data.txt` and `1500fN_maskedData.mp4` for each trial.
4. Run `plotData.m` — produces the comparison plot and `forward_1500_meanData.csv`.

**Required MATLAB toolboxes:** Image Processing Toolbox (for `VideoReader`, `VideoWriter`, color thresholding).
